# UI Application Instructions

## Project Rules

- All generated UI components must follow project conventions.
- Components go inside:
  `src/components/emerald-ui-components`

- Documentation goes inside:
  `src/content/docs/components`

- Registry updates must modify:
  `src/registry/registry-components.ts`

- Follow the implementation rules from:
  `.agent/skills/create_component_and_docs/SKILL.md`

## Git Workflow

- Always create a feature branch before modifications.
- Always commit completed work.

## Skills

Use the `import-gsap-animation` skill whenever asked to:
- import animations
- integrate GSAP/Motion demos
- convert HTML/CSS/JS animations into React components
