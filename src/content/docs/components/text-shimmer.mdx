## Props

| Prop        | Type                        | Default          | Description                   |
| ----------- | --------------------------- | ---------------- | ----------------------------- |
| `children`  | `React.ReactNode \| string` | `"Default Text"` | Text content to display       |
| `as`        | `React.ElementType`         | `"span"`         | HTML element to render as     |
| `duration`  | `number`                    | `4`              | Animation duration in seconds |
| `className` | `string`                    | `undefined`      | Additional CSS classes        |
| `...props`  | `HTMLAttributes`            | -                | All standard HTML attributes  |
