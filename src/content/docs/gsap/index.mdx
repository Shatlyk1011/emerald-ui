# GSAP Components

Welcome to the GSAP Components library! These components leverage the power of GSAP (GreenSock Animation Platform) for buttery-smooth, high-performance animations.

## Why GSAP?

GSAP is the industry-standard JavaScript animation library trusted by over 11 million websites. It offers:

- **Unmatched Performance**: Hardware-accelerated animations that consistently run at 60fps
- **Timeline Control**: Precise sequencing with `gsap.timeline()` for complex animation choreography
- **Browser Compatibility**: Works flawlessly across all browsers and devices
- **Advanced Features**: Morphing, dragging, physics-based animations, and scroll-triggered effects
- **Professional Grade**: Used by Apple, Google, Nike, and major brands worldwide
- **Plugin Ecosystem**: ScrollTrigger, Draggable, MorphSVG, MotionPath, and more

## When to Use GSAP Over Framer Motion

Choose GSAP components when you need:

- **Complex Timeline Sequences**: Multiple animations that need precise timing and orchestration
- **Scroll-Triggered Animations**: Hero sections, parallax effects, and scroll-based storytelling
- **Maximum Performance**: Handling many simultaneous animations without frame drops
- **SVG Animations**: Path morphing, drawing, and complex SVG manipulations
- **Framework Agnostic Code**: Animations that work across React, Vue, or vanilla JavaScript
- **Professional Projects**: Commercial websites requiring production-grade animation quality

> [!TIP]
> Not sure which animation library to use? Check out the [comprehensive comparison](/docs) in the main documentation to understand the differences between Framer Motion and GSAP.

## Getting Started

All components in this section use GSAP for their animations. Make sure you have GSAP installed:

```bash
pnpm install gsap @gsap/react
```
